You can find a proper writeup of the library [here](http://blog.tortuga22.com/2010/05/31/announcing-tortuga-22-ninepatch/).

To use this library, Cocoapods is prefered, add:

```
pod 'Tortuga22-NinePatch'
```

to your Podfile.
